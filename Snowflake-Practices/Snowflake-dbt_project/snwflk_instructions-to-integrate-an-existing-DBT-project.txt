--------------------In Snowflake an API integration -------------------------------
Which is an API that we use to integrate Snowflake with GitHub


CREATE AN API INTEGRATION IN SNOWFLAKE

1 open a new SQL file and type:

USE ROLE ACCOUNTADMIN;

CREATE OR REPLACE API INTEGRATION my_git_api_integration
        API_PROVIDER = git_https_api
        API_ALLOWED_PREFIXES = ( 'https://github.com/Albertbilu43' )
        API_USER_AUTHENTICATION = ( TYPE = SNOWFLAKE_GITHUB_APP )
        ENABLED = TRUE;
